# OutfitGo

Este proyecto se ha hecho con [Angular CLI](https://github.com/angular/angular-cli) version 16.2.16.

## Servidor de Desarrollo

Ejecuta `ng serve` para iniciar la aplicacion. Abre la siguiente direccion en tu navegador: `http://localhost:4200/`. La aplicación está diseñada para que cualquier cambio que hagas en los archivos de configuracion, se cambie a tiempo real..

## Creacion de componentes

Ejecuta `ng generate component component-name` para crear un nuevo componente.

## Build

Ejecuta `ng build`Para construir el proyecto. Los artefactos de la build se almacenan en el directorio `dist/`.
